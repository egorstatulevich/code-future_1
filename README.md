# Трекер прочитанных книг

Консольное приложение на Python для учета прочитанных книг.

## Возможности

- добавить книгу;
- показать список всех книг;
- рассчитать среднюю оценку;
- показать статистику по авторам;
- удалить книгу;
- сохранить данные в файл books.json.

## Поля книги

- автор;
- название;
- оценка от 1 до 5;
- дата прочтения.

## Запуск

```bash
python main.py
